Terms created for the RDF data model of academic and social impacts of the REALM framework
Term Description
Classes
SocialImpact Social impact of a researcher
AcademicImpact Academic impact of a researcher
Object Properties
hasSocialImpact It defines that a researcher has a social impact
hasAcademicImpact It defines that a researcher has a academic impact
Datatype Properties
socialMediaCitations Number of researcher's citations in social media
facebookCitations Number of researcher's citations on the online social network Facebook
gplusCitations Number of researcher's citations on the online social network Google+
spellingsUsed Alternative spellings to a researcher's name
subnetwork In a scientific social network, it refers to the subnetwork in which a researcher belongs
numberOfPublications Number of researchers' publications
betweenness Betweenness of a researcher in an scientific social network
closeness Closeness of a researcher in an scientific social network
degree Degree of a researcher in an scientific social network
pagerank Pagerank of a researcher in an scientific social network
betweennessPos In a ranking, it represents the position of a researcher in the betweenness
closenessPos In a ranking, it represents the position of a researcher in the closeness
degreePos In a ranking, it represents the position of a researcher in the degree
pagerankPos In a ranking, it represents the position of a researcher in the pagerank
score Value that represents the sum of a researcher's positions in predefined metrics
scoreAlt Social score of a researcher
hasNumberOfPublications Number of publications in a session
hasNumberOfElements Number of elements of a session
